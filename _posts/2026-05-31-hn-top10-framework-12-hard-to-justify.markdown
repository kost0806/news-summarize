### 개요

유명 테크 유튜버이자 개발자인 Jeff Geerling이 Framework Laptop 12를 직접 사용한 후 쓴 리뷰입니다. Framework는 수리 가능성(repairability)과 모듈식 설계로 사랑받는 브랜드이지만, 12인치 모델은 애플의 MacBook Neo와 비교했을 때 가격 대비 가치를 정당화하기 어렵다는 결론을 내립니다. 265점, 432개의 댓글을 기록했습니다.

### 주요 내용

#### Framework 12의 약점

Geerling은 Framework 12가 나쁜 노트북은 아니지만 "나쁜 가성비"라고 평가합니다. MacBook Neo와 비교한 구체적 단점은 다음과 같습니다:

- **성능**: 대부분의 벤치마크에서 MacBook Neo에 뒤처짐
- **효율성**: 배터리 및 발열 효율 모두 열세
- **소음**: Framework가 더 시끄러움
- **빌드 품질**: MacBook Neo가 더 견고하고 정제된 마감
- **디스플레이**: MacBook Neo의 화면 품질이 훨씬 우수
- **무게/두께**: 12인치임에도 MacBook Neo보다 무겁고 두꺼움

#### 디스플레이와 태블릿 모드의 타협

Framework 12는 360도 힌지를 적용해 태블릿 모드를 지원하지만, 이를 위해 화면 품질을 희생했습니다. 구형 스타일러스 기술을 사용해 iPad나 Surface의 최신 펜 경험에 크게 못 미칩니다. Geerling은 실제로 태블릿 모드를 써보니 노트북을 태블릿처럼 쓰는 것 자체가 불편하다는 사실을 깨달았다고 밝혔습니다.

#### Framework 12의 강점: 모듈식 포트

그나마 Framework 12가 빛나는 부분은 모듈식 확장 포트(expansion modules)입니다. 4개의 확장 모듈 슬롯에 USB-C, USB-A, HDMI, SD카드 등 원하는 포트를 자유롭게 구성할 수 있습니다. 각 포트에서 USB 3.2 Gen 2x1 성능을 지원합니다.

#### Framework 13이 더 나은 선택

Geerling은 Framework 팬이라면 12인치 대신 13인치 라인업을 고려하라고 권합니다. 수리 가능성, 업그레이드 가능성, 뛰어난 Linux 지원이라는 Framework의 장점은 13인치에서 더 잘 발휘된다는 평가입니다.

### HN 커뮤니티 반응

265점, 432개의 댓글로 Framework의 방향성에 대한 논쟁이 열렸습니다. Framework의 열성 팬들은 "수리 가능성과 업그레이드 가능성이 있어야 정말 내 소유의 기기"라며 가성비보다 가치를 강조했습니다. 반면 현실주의자들은 "MacBook Neo가 훨씬 더 나은 경험을 제공한다"고 지적했습니다. Framework의 ARM 프로세서 지원 로드맵에 대한 기대감도 댓글에서 나왔습니다.

### 시사점

Framework의 수리 가능성 철학은 여전히 의미 있지만, 12인치 모델은 그 철학을 성능과 가격 경쟁력 면에서 충분히 뒷받침하지 못했습니다. 환경을 생각하는 소비자와 Linux 사용자에게는 여전히 매력적인 선택지이지만, 단순히 "좋은 노트북"을 원한다면 MacBook Neo가 더 합리적인 선택입니다.
