### 개요

**Griffin PowerMate**는 2000년대 초반에 출시된 독특한 USB 다이얼 컨트롤러입니다. 단순히 돌리거나 누를 수 있는 금속 노브(knob)로, 파란 LED 조명이 내장되어 있습니다. 이 드라이버는 오래된 PowerMate 장치를 현대 macOS에서 다시 사용할 수 있게 해줍니다.

### Griffin PowerMate란?

- **외형**: 은색 알루미늄 원통형 노브에 파란 LED가 내장된 미니멀한 디자인
- **기능**: 돌리기(스크롤), 누르기(클릭), 길게 누르기(우클릭)
- **원래 용도**: 영상·음성 편집 시 스크롤 및 볼륨 조절 보조 입력 장치
- **슬로건**: "나의 USB 잡동사니 서랍에서 꺼내서 먼지를 털어내세요!"

### 이 드라이버의 특징

```
설치: DMG를 열고 PowerMate Agent를 Applications 폴더에 드래그
실행: PowerMate Agent 실행 후 메뉴바에서 설정 가능
```

**지원 기능**:
- **돌리기**: 활성 창 스크롤 또는 선택한 값 증감
- **스크롤 방향 반전** 가능
- **누르기**: 마우스 클릭
- **길게 누르기**: 우클릭
- **LED 밝기** 제어 (작동 상태에 따라 변경)

### 왜 주목받나?

현대 macOS(Apple Silicon 포함)는 2000년대 초반의 USB 장치를 대부분 지원하지 않습니다. 이 드라이버는 레트로 하드웨어를 사랑하는 사용자들이 소중한 장치를 버리지 않고 계속 사용할 수 있게 해줍니다.
